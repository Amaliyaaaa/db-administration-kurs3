# db-administration-kurs3
Целью этого курсового проекта была разработка автоматизированной системы сбора и обработки данных коммерческих организаций, включающая в себя реляционную базу данных, реализованную в среде MySQL, и программное обеспечение на языке C# с графическим интерфейсом Windows Forms.
При проектировании базы данных были применены принципы нормализации, реализованы связи между таблицами с помощью внешних ключей, а также предусмотрена масштабируемость структуры для возможного расширения в будущем. Были также разработаны тематические SQL-запросы, включая представления, CTE, процедуры, функции и триггеры.
Клиентская часть системы позволяет пользователям в удобной форме управлять информацией в зависимости от предоставленных прав доступа.
## Что реализовано:
1. Аутентификация и авторизация, хэширование паролей.
2. В базе данных хранятся данные о трех организациях, в каждой из которых от одного до трех пользователей с разными ролями. Все данные об организации разделены на категории, каждая категория отображается на отдельной вкладке приложения - простая и удобная навигация.
3. Полное управление базой данных через приложение: записи в каждой таблице (кроме самого перечня организаций) подлежат редактированию и удалению, есть функция добавления новых строк в таблицы.
4. Подключение к базе данных (MySQL) производится локально, я реализовала через localhost.
5. Проект выполнен при использовании средств Visual Studio 2022, платформа .NET, на C#.
### Как протестировать:
Скачать папку release, запустить exe-файл. Exe-файл можно найти по следующему пути: db_bi_forms/bin/Release/db_bi_forms.exe

Также необходимо создать локальную базу данных на вашем устройстве и при необходимости отредактировать параметры подключения к ней в коде (в случае внесения изменений в код проект запускать из среды разработки, а не exe-файл) - Form1.cs и Form2.cs содержат необходимые строки.
Для заполнения базы данных тестовой информацией добавила в репозиторий sql-файл.
